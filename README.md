# Real-Time Dynamic Pricing Analytics

## Project Overview
This project analyzes e-commerce transaction data to evaluate whether
dynamic pricing strategies increase revenue compared to static pricing.

## Business Problem
Can dynamic pricing based on time, demand, and customer behavior
increase overall revenue?

## Dataset
Source: Olist E-Commerce Dataset (Kaggle)

## Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Spyder IDE
- Git & GitHub

## Project Workflow
1. Data Cleaning
2. Data Merging
3. Feature Engineering
4. Dynamic Pricing Logic
5. Power BI Dashboard

## Key Insights
- Dynamic pricing increased revenue during peak hours
- Higher customer ratings correlated with higher dynamic prices
- Certain regions benefited more from dynamic pricing
